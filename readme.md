# SaliouStructCraft 
## Description
SaliouStructCraft is a library for creating and manipulating data structures in java with ease.
## Installation
### Maven
```xml
<dependency>
    <groupId>com.github.salioussalifou</groupId>
    <artifactId>salioustructcraft</artifactId>
    <version>1.0.0</version>
</dependency>
```
